# SpatialInsight

SpatialInsight is an end-to-end AI analytics platform for evaluating spatial entities using data-driven and explainable models.

This repository is organized as a monorepo:
- `ui/` for the frontend (Next.js)
- `platform/` for backend services and APIs
- `ml/` for machine learning (feature engineering, training, inference)

## Project Structure

```text
spatial-insight/
├── ui/          # Frontend application (Next.js)
├── platform/    # Backend services and APIs
├── ml/          # Machine learning module
```
